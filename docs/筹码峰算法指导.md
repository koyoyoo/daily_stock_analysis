筹码峰核心逻辑

在量化交易圈，这种算法被称为 **Volume Profile（成交量分布）**。商业软件（如同花顺）的“移动筹码分布”之所以看起来玄乎，是因为他们加入了一个“时间衰减因子”（即假设时间越久，老筹码被洗掉的概率越大）。

但对于实战操盘手来说，用**不带衰减的固定周期成交量分布**（比如看近 60 天或 90 天的真实成交堆积），往往能更精准地找到**绝对支撑位和压力位**。

它的计算原理非常直白，就三步：

1. **搭架子：** 找出这段时间的最高价和最低价，把中间的价格切成 50 或 100 个“等分抽屉”。
2. **切西瓜：** 遍历每一天的 K 线，看当天的最高价和最低价跨越了哪几个抽屉，把当天的成交量按比例塞进这些抽屉里。
3. **垒砖头：** 把所有天数的抽屉加总，里面成交量（砖头）最多的那个抽屉，就是**筹码单峰密集区（POC，Point of Control）**。

下面我为你写好了一段可以直接运行的 Python 代码，完全依赖 AkShare 免费接口。你可以直接把它塞进 `daily_stock_analysis` 项目的自定义脚本里：

```python
import akshare as ak
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

def calculate_chip_distribution(stock_code="002371", days=90, bins=50):
    """
    计算并绘制简易筹码峰 (Volume Profile)
    :param stock_code: 股票代码 (如 "002371" 北方华创)
    :param days: 计算过去多少天的筹码堆积
    :param bins: 价格切片数量 (抽屉数，越大越精细)
    """
    print(f"正在获取 {stock_code} 近 {days} 个交易日的行情数据...")

    # 1. 获取前复权历史日K线数据
    try:
        df = ak.stock_zh_a_hist(symbol=stock_code, period="daily", adjust="qfq")
        df = df.tail(days).reset_index(drop=True)
    except Exception as e:
        return f"获取数据失败: {e}"

    highs = df['最高'].astype(float)
    lows = df['最低'].astype(float)
    volumes = df['成交量'].astype(float)
    closes = df['收盘'].astype(float)

    # 2. 设定价格区间 (抽屉)
    min_price = lows.min()
    max_price = highs.max()
    # np.linspace 将最低价到最高价等分成 bins 个区间
    price_bins = np.linspace(min_price, max_price, bins + 1)
    volume_profile = np.zeros(bins)

    # 3. 核心切分算法：把每一天的成交量平摊到价格区间内
    for i in range(len(df)):
        h = highs.iloc[i]
        l = lows.iloc[i]
        v = volumes.iloc[i]

        # 如果遇到一字涨停/跌停，高低价相等，直接把量算在对应的抽屉里
        if h == l:
            idx = np.digitize(h, price_bins) - 1
            idx = min(max(idx, 0), bins - 1)
            volume_profile[idx] += v
            continue

        # 如果有波动，计算当天价格跨越了哪些抽屉，按比例分发成交量
        for j in range(bins):
            bin_lower = price_bins[j]
            bin_upper = price_bins[j+1]

            # 计算当天的K线柱体和当前抽屉的重合部分
            overlap_lower = max(l, bin_lower)
            overlap_upper = min(h, bin_upper)

            if overlap_upper > overlap_lower:
                # 将当天的成交量按重合比例分配
                ratio = (overlap_upper - overlap_lower) / (h - l)
                volume_profile[j] += v * ratio

    # 4. 寻找筹码最密集的区域 (POC - Point of Control)
    max_vol_idx = np.argmax(volume_profile)
    poc_price = (price_bins[max_vol_idx] + price_bins[max_vol_idx + 1]) / 2
    current_price = closes.iloc[-1]

    print("-" * 30)
    print(f"当前最新收盘价: {current_price:.2f} 元")
    print(f"核心筹码密集区 (支撑/压力位): 大约在 {poc_price:.2f} 元附近")

    # 判断当前筹码状态
    if current_price > poc_price:
        print("结论: 股价在筹码峰上方，获利盘较多，下方筹码峰形成【支撑】。")
    else:
        print("结论: 股价跌破筹码峰，上方套牢盘沉重，上方筹码峰形成【强压】。")

    # 5. 可视化绘图 (画出专业的 Volume Profile)
    plt.rcParams['font.sans-serif'] = ['SimHei'] # 解决中文显示问题
    plt.rcParams['axes.unicode_minus'] = False

    fig, ax1 = plt.subplots(figsize=(10, 6))

    # 右侧画横向筹码峰
    y_pos = (price_bins[:-1] + price_bins[1:]) / 2
    ax1.barh(y_pos, volume_profile, height=(max_price-min_price)/bins*0.8, color='grey', alpha=0.5, label='成交量堆积')
    ax1.axhline(poc_price, color='red', linestyle='--', linewidth=2, label=f'POC筹码核心: {poc_price:.2f}')

    # 为了直观，把最新价格也画上去
    ax1.axhline(current_price, color='blue', linestyle='-', linewidth=2, label=f'当前价格: {current_price:.2f}')

    ax1.set_title(f"{stock_code} 近 {days} 日筹码分布 (Volume Profile)")
    ax1.set_xlabel("累积成交量")
    ax1.set_ylabel("价格 (元)")
    ax1.legend()
    plt.grid(True, alpha=0.2)
    plt.show()
```
